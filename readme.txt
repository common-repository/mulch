== Description ==
Mulch is a content activation and recommendation platform that helps marketers better understand buyers using zero-party and third-party data, automatically target them with personalized content, and move them through the sales funnel faster. Mulch is a part of Reve Marketing, a data-driven marketing platform that powers personalized experiences for customer acquisition and engagement.

* Personalized greeting

== Installation ==
= AUTOMATIC INSTALLATION THROUGH WORDPRESS: =

1. In your WordPress admin sidebar, go to Plugins and then click on Add New
2. Search for Mulch
3. Click Install Now and activate the plugin


= MANUAL INSTALLATION VIA FTP: =

1. Go to https://wordpress.org/plugins/mulch and click on Download to download the Mulch zipped plugin files
2. Unzip the plugin files
3. In FTP, upload the entire mulch plugin folder you’ve just unzipped to your/wp-content/plugins directory
4. Go back to your WordPress dashboard, click on Plugins in the sidebar, find the Mulch plugin you’ve just uploaded and click Activate


= How it works: =
1. After installing Mulch plugin, register with Mulch first
2. Feed content assets to Mulch - Mulch can also scrape your website and can identify content assets or
3. Publish Mulch to your Wordpress blog / site.
4. Later you can create more content panels and assign rules and enable progressive profiling from the dashboard.
